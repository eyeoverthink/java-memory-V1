# Calculator

> Perform mathematical calculations and evaluate expressions

## Usage

Use this skill to calculate mathematical expressions, solve equations, or perform numerical analysis.

## Syntax

```
TOOL:CALC <expression>
```

## Examples

```
TOOL:CALC 2 + 2
TOOL:CALC sqrt(144)
TOOL:CALC (5 * 8) / 2
TOOL:CALC sin(pi/2)
```

## Capabilities

- Basic arithmetic: +, -, *, /
- Advanced functions: sqrt, sin, cos, tan, log, exp
- Constants: pi, e
- Parentheses for order of operations

## Safety

This tool is safe to use and runs in a sandboxed environment.
