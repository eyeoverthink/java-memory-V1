# Web Search

> Search the internet for information using various search engines

## Usage

Use this skill to find current information, research topics, or verify facts from the web.

## Syntax

```
TOOL:SEARCH <query>
```

## Examples

```
TOOL:SEARCH latest AI developments 2026
TOOL:SEARCH quantum computing breakthroughs
TOOL:SEARCH golden ratio in nature
```

## Capabilities

- Google search
- DuckDuckGo search
- Wikipedia lookup
- News search
- Academic paper search

## Safety

Web searches are read-only and do not modify any data. Results are fetched and presented to the user.

## Implementation

This skill requires internet access and appropriate API keys or web scraping capabilities.
