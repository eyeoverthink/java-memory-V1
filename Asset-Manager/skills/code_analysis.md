# Code Analysis

> Analyze code structure, detect patterns, and suggest improvements

## Usage

Use this skill to understand code, find bugs, optimize performance, or learn programming concepts.

## Syntax

```
TOOL:ANALYZE <code_snippet>
```

## Examples

```
TOOL:ANALYZE public class Test { public static void main(String[] args) { System.out.println("Hello"); } }
```

## Capabilities

- Syntax checking
- Code complexity analysis
- Performance optimization suggestions
- Security vulnerability detection
- Best practices recommendations
- Design pattern identification

## Safety

Code analysis is read-only and does not execute or modify code.
